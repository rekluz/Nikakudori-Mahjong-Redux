# Nikakudori Mahjong

A beautiful and peaceful Japanese tile-matching solitaire game. Clear the board by matching identical pairs to reveal scenic images of Japan and inspirational proverbs.

## What's New (v5.5.2)

- **Pinch-to-Zoom** - Pinch gestures to zoom the game board
- **Refactored Game Engine** - Better code organization with dedicated use cases
- **3D Board Improvements** - Fixed tile placement and generation for layered layouts
- **Welcome Screen** - New animated welcome screen with visual effects
- **Localization** - Additional language strings (ES, FR, IT, TL)
- **Bug Fixes** - Various tweaks and improvements to game stability

## How to Play

1. **Objective:** Remove all tiles by matching identical pairs
2. **Selectable Tiles:** Must have at least one free side (left/right) and no tile on top
3. **Making Matches:** Tap two matching free tiles to remove them
4. **Path Rule:** Tiles connect with max 2 bends
5. **Winning:** Clear the board to win

## Features

- **Two Game Modes:** Regular and Gravity mode
- **Four Difficulties:** Easy, Normal, Hard, Extreme
- **6 3D Board Layouts:** Pyramid, Fortress, Turtle, Bridge, Dragon, Castle
- **Hint, Undo & Shuffle:** Powerful tools to overcome difficult positions
- **Timer & High Scores:** Track your best times
- **Multi-language:** Japanese, Tagalog, Spanish, French, Italian
- **Customizable:** Music, sound, vibration, zoom
- **Visual Effects:** Animated connections, particle bursts, scenic reveals
- **No Ads or Tracking:** Clean, stress-free gameplay
